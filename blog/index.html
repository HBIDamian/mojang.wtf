Minecraft Bedrock Edition

Windows 10, Apple Smart devices, android, play station, xbox, nintendo and so on so forth

Although the Minecraft Bedrock Edition fanbase is smaller than it's Minecraft Java Editon counterpart, it has a strong dedicated fanbase.
Despite this, there are a lot of issues with the game that the community feels like it takes away from the beauty and perfection of the game.

Bugs

Minecraft: Bedrock Edition is synonymous for having bugs that some consider to be gamebreaking.

  • A user by the name of Hatchibombotar (who creates add-ons for the game) has noted that the game is buggy, extremely laggy.
    They note issues from MCBE Version 1.13 with their issues. Go check it out at https://hatchibombotar.com/bedrock_state
